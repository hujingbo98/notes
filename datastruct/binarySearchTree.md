# 二叉搜索树

什么是二叉搜索树？

定义：对于二叉搜索树中的任意结点 x，其左子树中的关键字不超过 x 的关键字，其右子树中的关键字不小于 x 的关键字。

不同的二叉搜索树可以代表同一组值的集合，大部分的二叉搜索树操作的最坏运行时间与数的高度成正比。
